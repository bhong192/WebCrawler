    Accord des verbes pronominaux


    Quelques rappels :

    Malgré l'utilisation du verbe être, les verbes pronominaux ne s'accordent pas toujours.

    Le participe passé s'accorde avec le sujet si :

    1)  le sujet réalise l'action lui-même.

    Ex. : Elles se sont coiffées. 

    2)  le complément d'objet direct  (C.O.D.) est placé avant le verbe :

    Ex. :  Les messages qu'elles se sont envoyés. 

            Les vidéos qu'ils se sont partagées. 


    Le participe passé ne s'accorde pas avec le sujet  si :

    3)  le C.O.D. suit le verbe :

    Ex. :  Elles se sont brossé les cheveux. 

    4)  le verbe pronominal fonctionne avec un complément d'objet indirect (C.O.I.) et de ce fait l'action se réfléchit sur le sujet .

    Ex. :  Elles se sont plu. (à elles-mêmes)

    Cours complémentaires : -

     


    À l'aide du menu déroulant, accordez le verbe pronominal.
