    Futur (les 3 groupes)


    Pour rappel, les verbes des trois groupes ont les mêmes terminaisons au futur :

     

     

    1re personne (sg) : -ai            2e personne (sg) : -as             3e personne (sg) : -a (pas de S)

    1re personne (pl) : -ons          2e personne (pl) : -ez              3e personne (pl) : -ont

    Les verbes du 1er groupe se terminent par -er à l'infinitif.

    Les verbes en -ir du 2e groupe font leur participe présent en -issant.

    Les verbes du 3e groupe sont constitués par les autres verbes qui n'appartiennent pas aux deux premiers groupes.

    Il existe des verbes dont le radical se transforme au futur, parfois simplement par le doublement d'une consonne :

    Ex. : appeler : j'appellerai ; venir : je viendrai ; avoir : j'aurai ; être : je serai ; acquérir : j'acquerrai.

     

     

    Conjuguez les verbes suivants au futur.
