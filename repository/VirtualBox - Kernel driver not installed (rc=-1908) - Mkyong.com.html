VirtualBox – Kernel driver not installed (rc=-1908)

By | Last updated: June 11, 2021

Viewed: 13,250 (+1,728 pv/w)

Tags: | |

Try to start the VirtualBox on macOS and hits the below error?

P.S Tested with VirtualBox 6.1 on macOS Big Sur 11.1

1. Solution

The reason for this Kernel driver not installed (rc=-1908) is similar to this . The macOS blocked the VirtualBox to run, and we need to unblock it in the macOS Security & Privacy configuration screen.

Steps to allow the VirtualBox to run:

  1. Clicks the top-left Apple icon
  2. System Preferences...
  3. Security & Privacy
  4. If the left-bottom icon is locked, unlock it with a password.
  5. Clicks "Allow" for the developer Oracle America, Inc.
  6. Done, restart the macOS.

To fix it, clicks the "Allow" button for this System software from developer "Oracle America, Inc" has been updated
